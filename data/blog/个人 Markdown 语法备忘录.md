# 个人 Markdown 语法备忘录

## 1. 标题 (Headings)

在文字前添加 `#` 号和空格来创建标题。`#` 的数量代表标题的级别。

Markdown

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

## 2. 文本格式 (Text Formatting)

使用特定的符号将需要强调的文本包围起来。

| **语法**       | **描述**              | **效果**     |
| -------------- | --------------------- | ------------ |
| `**加粗文本**` | 粗体                  | **加粗文本** |
| `*斜体文本*`   | 斜体                  | _斜体文本_   |
| `***粗斜体***` | 粗斜体                | **_粗斜体_** |
| `~~删除线~~`   | 删除线                | ~~删除线~~   |
| `==高亮文本==` | 高亮 (部分编辑器支持) | ==高亮文本== |

---

## 3. 列表 (Lists)

### 无序列表

使用 `-`、`+` 或 `*` 加空格作为列表标记。

Markdown

```
- 项目一
- 项目二
  - 子项目 (前面加两个空格缩进)
```

### 有序列表

使用数字加英文句号 `.` 加空格。

Markdown

```
1. 第一步
2. 第二步
3. 第三步
```

### 任务列表

使用方括号 `[ ]` 表示未完成，`[x]` 表示已完成。

Markdown

```
- [x] 已经完成的任务
- [ ] 待完成的任务
```

## 4. 链接与图片 (Links & Images)

两者语法非常相似，图片只需在最前面加一个感叹号 `!`。

| **类型**     | **语法示例**                      |     |
| ------------ | --------------------------------- | --- |
| **超链接**   | `[谷歌](https://www.google.com)`  |     |
| **图片**     | `![替代文字](图片链接或本地路径)` |     |
| **快速链接** | `<https://www.google.com>`        |     |
|              |                                   |     |
|              |                                   |     |

```
![测试图片](static/images/Mikoto.png)
```

## 5. 代码 (Code)

### 行内代码

使用单个反引号 `` ` `` 将代码包围。

**语法：** 请使用 `` `Ctrl + C` `` 进行复制。

**效果：** 请使用 `Ctrl + C` 进行复制。

### 代码块

使用三个反引号 ` ``` ` 包围代码，并在第一行末尾指定编程语言（用于语法高亮）。

**语法：**

Plaintext

````
    ```python
    def hello_world():
        print("Hello, Markdown!")
    ```
````

## 6. 引用 (Blockquotes)

在段落前添加 `>` 和空格即可创建块引用。可以嵌套使用。

**语法：**

Markdown

```
> 这是一段引用。
>> 这是嵌套的引用。
```

**效果：**

> 这是一段引用。
>
> > 这是嵌套的引用。

## 7. 表格 (Tables)

使用竖线 `|` 分隔不同的列，使用连字符 `-` 分隔表头和内容行。可以通过冒号 `:` 控制对齐方式。

**语法：**

Markdown

```
| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 文本 | 文本 | 文本 |
| Markdown | 备忘录 | 完成 |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |

| :--- | :---: | ---: |

| 文本 | 文本 | 文本 |

| Markdown | 备忘录 | 完成 |

---

## 8. 分隔线 (Horizontal Rules)

在一行中使用三个或更多的星号 `***`、减号 `---` 或底线 `___` 即可生成一条分隔线（上方需留一空行）。

Markdown

```
---
```
